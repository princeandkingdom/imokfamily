How it works

Set up in minutes. Stay connected every day.

The older adult uses IMOK. Family members use IMOK Family. A simple QR code securely connects them.

  1. 1

    Install both apps

    Install IMOK for the older adult and IMOK Family for relatives or caregivers.

  2. 2

    Scan the QR code

    Use IMOK Family to scan the code shown in the older adult's app.

  3. 3

    Stay connected

    Receive safety alerts and view the latest available status in one place.

Common questions

Helpful answers before you begin.

Do I need both apps?

Yes. IMOK is used by the older adult, while IMOK Family is used by relatives or caregivers.

Can Android and iPhone users connect?

The service is designed for families using supported Android and iOS devices. Availability can vary by app version and region.

Does IMOK replace emergency services?

No. IMOK is a family safety and notification tool. In an emergency, contact local emergency services directly.

Does the app require internet access?

Yes. Location updates, alerts and cloud synchronization require a working internet connection.
